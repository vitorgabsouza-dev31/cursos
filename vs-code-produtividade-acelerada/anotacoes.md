# 🚀 Guia Definitivo: VS Code (Atalhos & Extensões)

## 📂 1. Navegação e Gerenciamento de Arquivos

- **`Ctrl + P`**: Busca rápida de arquivos (Go to File).
- **`Ctrl + Shift + P`**: Abre a **Paleta de Comandos** (Acesso a todas as funções do editor).
- **`Ctrl + B`**: Alterna a visibilidade da **Barra Lateral**.
- **`Ctrl + 0`**: Foca no **Explorador de Arquivos**. Navegue com as `Setas` e abra com `Enter`.
- **`Ctrl + '`** (ou `Ctrl + J`): Alterna a visibilidade do **Terminal Integrado**.
- **Estrutura Dinâmica**: Ao criar um arquivo, use `caminho/do/arquivo.ext` para gerar as pastas automaticamente.

## ✍️ 2. Edição e Produtividade Neutra

- **`Ctrl + Setas`**: Salta o cursor palavra por palavra.
- **`Shift + Setas`**: Seleção de caracteres (Combine com `Ctrl` para selecionar palavras inteiras).
- **`Ctrl + L`**: Seleciona a linha inteira.
- **`Ctrl + D`**: Seleciona a próxima ocorrência da palavra sob o cursor (Multi-cursor seletivo).
- **`Alt + Clique`**: Cria múltiplos cursores em locais arbitrários.
- **`Alt + Setas (↑/↓)`**: Move a linha ou bloco selecionado para cima ou para baixo.
- **`Ctrl + /`**: Comenta ou retira o comentário da linha/bloco.
- **`Ctrl + C / V`** (Início da linha): Copia e cola a linha inteira, sem precisar selecioná-la.

## 🧩 3. Extensões Essenciais (Power-ups)

### 🛠️ Desenvolvimento & Debug

- **Live Server**: Servidor local com _auto-reload_ para testes em tempo real (HTML/CSS/JS).
- **Code Runner**: Executa trechos de código de diversas linguagens instantaneamente.
- **Console Ninja**: Exibe os logs do `console.log` diretamente no editor, ao lado do código.
- **Error Lens**: Destaca erros e avisos de sintaxe diretamente na linha onde ocorrem.
- **WSL**: Integração nativa para desenvolver dentro do ambiente Linux no Windows.

### 🐘 Ecossistema PHP

- **PHP Intelephense**: A melhor ferramenta de _IntelliSense_ (autocompletar e diagnóstico) para PHP.
- **PHP Awesome Snippets**: Coleção de atalhos de código para agilizar a escrita de funções PHP.

### 📜 Organização & Estética

- **Prettier**: Formatador de código que garante um padrão visual limpo e consistente.
- **Better Comments**: Melhora a legibilidade categorizando comentários (Alertas, Dúvidas, TODOs).
- **Auto Rename Tag**: Renomeia automaticamente a tag de fechamento ao alterar a de abertura.
- **Polacode**: Gera capturas de tela elegantes do seu código para compartilhamento.
- **Bookmarks**: Permite marcar linhas específicas para navegação rápida em arquivos longos.

### 🌿 Git & Versionamento

- **GitLens**: Exibe quem alterou cada linha (Git Blame) e o histórico de commits no próprio editor.
- **Git History**: Interface visual completa para visualizar o log e o grafo de commits.

## ⚙️ Dicas de Configuração (Paleta de Comandos)

Para acessar estas funções, use `Ctrl + Shift + P`:

1. **"Preferences: Open User Settings (JSON)"**: Abre o arquivo de configuração bruta do VS Code. É onde você cola configurações avançadas de temas e fontes.
2. **"Emmet: Wrap with Abbreviation" (Envolver com Abreviação)**: Selecione um bloco de código e use este comando para envolvê-lo em uma nova tag HTML (ex: uma `div` ou `section`).
3. **"Configure Display Language"**: Use para alternar entre Inglês e o **Portuguese (Brazil) Language Pack**.
